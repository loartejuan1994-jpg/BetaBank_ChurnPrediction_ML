# ML-BetaBank-ChurnPrediction
Modelo de clasificación binaria para predecir abandono de clientes (churn) en Beta Bank.  Se aplicaron técnicas de corrección de desequilibrio de clases (SMOTE y Oversampling)  logrando F1 Score ≥ 0.59 con Random Forest. Incluye análisis EDA, comparativa de modelos  y métricas AUC-ROC.
